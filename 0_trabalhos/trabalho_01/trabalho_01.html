Perguntas sobre HTML

Nelson Abrantes

O que é HTML? E para que serve?

HTML(HyperText Markup Language) é uma linguagem de marcação utilizada na construção de páginas web.
E serve para estruturar e definir o conteudo de uma página web.

Que ferramentas são necessárias para construir um site em HTML?

Um editor de código é o software necessário para construir um site em html,
mas qualquer editor de texto serve, como por exemplo o Bloco de notas.

Quais os elementos base necessários para a criação de uma página web em HTML?

Os elementos necessários são:
<!DOCTYPE html>
<html></html>
<head></head>
<body></body>

O que é um elemento? Dê exemplos.

Um elemento é constituído por a tag de abertura, a tag de fechamento e o conteúdo.
Um exemplo de um elemento é <h1>Isto é um titulo<h1>.

O elemento <br> necessita de ser fechado? Justifique.

O elemento <br> não necessita de ser fechado porque é um elemento vazio.
Os elementos vazios são de "fechamento automático" e são usados para introduzir imagens, quebras, linhas horizontais etc.

Para que servem os atributos? Dê exemplos.

Os atributos servem adicionar informação extra sobre o elemento, sem que apareça no conteudo.
Um exemplo de um atributo é <meta charset="UTF-8"> neste caso o sublinhado é o atributo.
